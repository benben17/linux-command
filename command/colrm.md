colrm
===

Remove specific columns from files

## Description

The **colrm command** is used to remove specified columns from a file. It reads from standard input and writes to standard output. Without any parameters, the `colrm` command does not filter any content.

### Syntax

```shell
colrm [arguments]
```

### Arguments

*   Start Column: The column number at which to start removing.
*   End Column: The column number at which to stop removing.
