xlsclients
===

List client applications running on a display.

## Supplemental Information

The **xlsclients command** is used to list client applications running on a display.

### Syntax

```shell
xlsclients(options)
```

### Options

```shell
-a: List client application information for all displays.
-display <display_id>: Specify the display number for connecting to the X Server, starting from "0" and incrementing sequentially.
-l: Use long format listing.
-m <max_command_length>: Set the maximum length of display command information, in characters.
```
