ip6tables-save
===

Save ip6tables table configurations.

## Description

The `ip6tables-save` command dumps the contents of an IPv6 table in an easily parseable format to STDOUT.

### Syntax

```shell
ip6tables-save [options]
```

### Options

```shell
-c : Include the current values of all packet and byte counters in the output.
-t : Specify the name of the table to save.
```
