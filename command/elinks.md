elinks
===

Text-mode WWW browser.

## Description

The **elinks** command provides a text-mode WWW browser, similar in operation to "lynx".

### Syntax

```shell
elinks [options] [parameters]
```

### Options

```shell
-anonymous: Use anonymous account mode.
-auto-submit: Automatically submit the first form encountered.
-config-dir <dir>: Specify the directory for elinks configuration and runtime state.
-dump: Print the HTML document in plain text to standard output.
-version: Display version information.
-h, --help: Display help information.
```

### Parameters

URL: Specify the URL to visit.
