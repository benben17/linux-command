ftptop
===

Display the connection status of the ProFTPD server.

## Description

The **ftptop command** displays the connection status of the ProFTPD server in a style similar to the `top` command.

### Syntax

```shell
ftptop (options)
```

### Options

```shell
-D: Filter out sessions that are currently downloading.
-S: Display connection status only for the specified virtual host.
-d: Specify the screen refresh interval.
```
