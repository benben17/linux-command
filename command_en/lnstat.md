lnstat
===

Display the network status of the Linux system

## Description

The **lnstat** command is used to display the network status of the Linux system.

### Syntax

```shell
lnstat (options)
```

### Options

```shell
-h: Display help information;
-V: Display version information;
-c: Specify the number of times to display network status at intervals;
-d: List available files or keys;
-i: Specify the interval in seconds between updates;
-k: Display only the specified keys;
-s: Toggle display of the header;
-w: Specify the width of each field.
```
